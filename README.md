# Ebook catalog API

## How to run

### Windows

#### JAR
1. mvnw.cmd clean package
2. java -jar .\target\ebookapi-1.0.0.jar

#### Spring Boot Maven Plugin
- mvnw.cmd spring-boot:run


### Linux

#### JAR
1. mvnw clean package
2. java -jar ./target/ebookapi-1.0.0.jar

#### Spring Boot Maven Plugin
- mvnw spring-boot:run
