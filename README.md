# Ebook catalog API

## How to run

### Windows

#### JAR
1. .\mvnw clean package
2. java -jar .\target\ebookapi-1.0.0.jar

#### Spring Boot Maven Plugin
- .\mvnw spring-boot:run


### Linux

#### JAR
1. mvn clean package
2. java -jar .\target\ebookapi-1.0.0.jar

#### Spring Boot Maven Plugin
- mvn spring-boot:run
